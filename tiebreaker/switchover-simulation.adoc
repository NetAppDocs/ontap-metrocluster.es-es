= Simular un cambio usando MetroCluster Tiebreaker
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
A partir de MetroCluster Tiebreaker 1.7, puede simular un cambio para probar la funcionalidad de cambio de Tiebreaker.

.Acerca de esta tarea
* El `monitor switchover-simulate` El comando solo es compatible con Tiebreaker 1.7 o posterior.


.Pasos
. Verifique el estado de monitoreo del desempate:
`monitor show -status`

. Realizar una simulación de conmutación:
+
`monitor switchover-simulate -monitor-name <monitor_name> -cluster <cluster_name>`

+
El comando devuelve la siguiente salida para una operación exitosa:

----
